# Mission Launches EDA

# Source of the Dataset
Originally obtained from Kaggle:
https://www.kaggle.com/datasets/sefercanapaydn/mission-launches

## Description of the Dataset
This dataset includes information regarding 1957 space missions till present, it conveys features such as organisation, location, date, rocket status, mission status, and the price.

# The overall number of rows and columns of information
- Rows: 4324
- Columns: 9 

## Description of Features
- Organization: the corporation or agency in charge of the launch
- Location: of the launch point
- Date: The mission's date and time
- Detail: Mission's description
- Rocket_Status: The rocket's current state (Active/Retired)
- Mission_Status: The mission's result (success, failure, etc.)
- Price: Launch cost (contains missing values) 

## Dataset's objective
With the modern focus on the space industry and it's rapid advancements, it's essential to recognize patterns, find trend, get to know success rates, the distribution across countries and corporations.
